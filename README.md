## What I'm currently working on

## 🧠 Master Thesis

I'm designing and developing a **VS Code extension** that helps developers keep control when working with AI-assisted coding tools.

The goal isn't to replace AI, it's to make collaboration between developer and AI more transparent and intentional.

Planned stack:

Extension: TypeScript (VS Code API)  
Backend: Node.js + Express  
Database: Neon (serverless Postgres)  
ORM: Prisma  
Webview UI: React  
Deployment: Fly.io (Docker)  
Bundler: Vite

   
## A few projects

## ✅ ToDo (Next.js + Supabase)

A full-stack todo application built to explore modern web development patterns.

Frontend: Next.js, React, TypeScript, Tailwind  
Backend: Next.js API routes  
Database & Auth: Supabase (PostgreSQL)  
Testing: Playwright  
Deployment: Vercel

This project was mainly about understanding full-stack architecture in practice: authentication, database design, API routes and testing.



## ✨ StarGazer 

A web app for people who like astronomy and travel. You can search for a location and get weather forecasts, the current moon phase, astronomy news and constellation information. Users can save places they've visited or want to visit and keep track of constellations seen at those locations.

Frontend: Vue 3, Vite, Bootstrap  
Backend / Services: Firebase  
Database: Firestore  
External APIs: GeoDB Cities, WeatherAPI, Google News, Moon Phase API, Stars API

This project was a fun mix of data integration and frontend architecture.



## 📊 Big Data Sentiment Analysis


A data science project analyzing public opinion about generative AI on social media.

Language: Python  
Data Processing: Pandas, NumPy  
NLP / Models: Hugging Face Transformers, PyTorch  
Analysis & Visualization: Scikit-learn, Matplotlib, Seaborn

The final dataset contains ~490k cleaned tweets about AI.

What I found interesting here was not just the modeling, but the process of making messy data usable.
